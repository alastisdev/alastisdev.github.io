About the Studio

SOLO DEVELOPMENT // SUNDERLAND, UK

A/D
Alastis Dev is a one-person indie studio building dark fantasy mobile games with engineering precision. Founded by a developer balancing autism, depression and full-time fatherhood — every line of code is written between school runs and bedtime stories. The workflow combines GameMaker Studio 2 with AI-augmented development tools (Claude, Gemini, NotebookLM) to compete with bigger studios on a one-person budget. The mission: prove that limitations breed creativity, and that mobile games can be both atmospheric and accessible.
